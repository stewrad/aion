# EE448 Advanced Wireless Communications Engineering - Optimization of Adaptive Modulation and Coding with Reinforcement Learning

A Python monorepo with multiple packages for building an Adapative Coding and Modulation simulation environment. The intent is to develop an ACM communications link and implement reinforcement learning (RL) algorithms to improve link efficiency.

## Features
- Core DSP algorithms (e.g., filtering, transforms, windowing)
- Learning development tools for FFTs/DFTs, Power Spectral Density (PSD), Amplitude, Phase, and Time plots

## Prerequisites
- Python 3.7 or higher
- [NumPy](https://numpy.org/) 

## Installation
Clone the repository and install any dependencies:

```bash
git clone https://github.com/stewrad/aion.git
cd aion
python -m venv .venv && source .venv/bin/activate
pip install --upgrade pip
pip install -e .[dev]
```
